To run the program:

Open Command Window at the Phylo directory
Execute command java -jar "dist/Phylo.jar"
Enter the number of dataset after the prompt appear(e.g: 1,2,3,..6)
Enter 'm' for memetic algorithm solution or s for scatter search solution.

The program can also be run by opeing the project in any IDE.

To compile the program:
Open the project in an IDE and perform clean and build
